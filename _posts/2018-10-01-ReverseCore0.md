OllyDgb基本指令  
{% highlight text %}
Restart						Crtl+F2  
Step Into					F7  
Step Over 					F8  
Execute till Return				Crtl+F9  
Execute till Cursor				F4  
Goto						Crtl+G
Comment						;  
Label 						:  
Set/Reset BreakPoint 				F2  
Run 						F9
Show the current EIP				*  
Show the previous EIP				-
Preview CALL/JMP address 			Enter
{% endhighlight %}
